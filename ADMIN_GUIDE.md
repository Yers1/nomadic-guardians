# Руководство администратора

## Вход в админ-панель

1. Перейдите на страницу **Admin** в навигации сайта
2. Введите пароль: `ramazantop1mentor`
3. Нажмите "Войти / Login"

## Управление спонсором

В админ-панели вы можете:
- Добавить/изменить название спонсора
- Добавить/изменить описание спонсора и его вклад
- Загрузить логотип спонсора (изображение)

Данные спонсора будут автоматически отображаться на главной странице сайта.

## Управление файлами команды

1. Перейдите на страницу **Файлы / Files**
2. Если вы авторизованы как админ, вы увидите кнопки для загрузки файлов
3. Вы можете загрузить следующие файлы:
   - **Technical Design Report** (Обязательный)
   - **Team Introduction** (Обязательный)
   - **Real-World Application Poster** (Опциональный)
   - **Community Outreach** (Опциональный)

4. После загрузки файлы будут доступны для скачивания всем посетителям сайта

## Выбор языка при первом посещении

При первом посещении сайта (или после очистки localStorage) пользователи увидят модальное окно выбора языка:
- English
- Русский
- Қазақша

После выбора язык сохраняется в браузере пользователя.

## Безопасность

- Пароль хранится в коде (для production рекомендуется использовать более безопасный метод аутентификации)
- Сессия админа хранится в sessionStorage браузера
- При закрытии браузера сессия завершается

## Структура данных

Данные сохраняются в:
- `/data/sponsor.json` - данные спонсора
- `/data/files.json` - информация о загруженных файлах
- `/public/uploads/` - загруженные файлы и изображения
